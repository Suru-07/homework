HTML- Hypertext Markup Language

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses a system of tags to structure content like text, images, and links.

More:

Features:

1)Simple and Easy to Learn – Uses straightforward tags to structure content.
2)Platform Independent – Works on all devices and browsers.
